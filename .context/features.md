- List of notes
- Filter by folder
- search by title

### Main

- Text editor
- Edit title and folder

### Topbar

- Left: button sync and sync alert
- Toolbar

- Empty state

- Title bar: "workspace.title" if available
- Optimistic update

- Sync screen: wait until local data loaded
