# Features

## Layout

- Left: sidebar (collapsible)
- Right: main content manager
- Topbar full width

### Sidebar

- List of notes
- Filter by folder
- Search by title
- User can pin notes
- User can click read-only hover effect (render markdown format)

### Main

- Text editor
- Edit title and folder

### Topbar

- Left: button sync and sync alert
- Toolbar

## UI Interaction

- Empty state
- Loading state
- Theme change: "workspace.theme"
- Darkmode: "workspace.darkmode"
- Title bar: "workspace.title" if available
- Optimistic update
- Optimistic rollback
- Splash screen: wait until local data loaded

## Popup

- Manage folder
- Setting persolaize
